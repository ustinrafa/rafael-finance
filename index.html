Google Sites sangat cocok untuk pemula dan tidak memerlukan pengaturan hosting yang rumit seperti Netlify. Karena aplikasi Anda dibangun menggunakan HTML, CSS, dan JavaScript murni, Anda bisa memasukkannya ke Google Sites menggunakan fitur "Sematkan Kode" (Embed Code). Berikut adalah langkah-langkah detail untuk memasang aplikasi keuangan Anda ke Google Sites: Langkah 1: Buat Situs Baru Buka Google Sites dan pastikan Anda sudah login dengan akun Google Anda. Klik tombol "+" (Situs Kosong / Blank) untuk membuat halaman baru. Agar aplikasi Anda terlihat penuh (seperti aplikasi sungguhan), arahkan kursor ke bagian Header (bagian atas yang ada teks judulnya), lalu klik ikon tempat sampah (Hapus header) di sebelah kiri. Langkah 2: Masukkan Kode Aplikasi Di menu sebelah kanan, klik tab Sisipkan (Insert). Pilih menu Sematkan (Embed) (ikonnya berbentuk tanda kurung sudut < >). Pada pop-up yang muncul, pilih tab Sematkan kode (Embed code). Buka file HTML terakhir yang Anda miliki, copy (salin) seluruh kodenya dari baris pertama hingga terakhir, lalu paste (tempel) ke dalam kotak kode di Google Sites tersebut. (Pastikan Anda menyalin kode yang sudah tidak ada tulisan "(rafael)" dan "(231420)"-nya). Klik Berikutnya (Next), lalu akan muncul pratinjau aplikasi Anda. Klik Sisipkan (Insert). Langkah 3: Sesuaikan Ukuran Layar Aplikasi Anda sekarang akan muncul di dalam sebuah kotak di halaman Google Sites. Klik kotak aplikasi tersebut, lalu tarik titik-titik biru di ujungnya untuk memperlebar dan memperpanjang kotaknya hingga memenuhi seluruh halaman putih yang ada. Pastikan ukurannya cukup panjang agar saat Anda masuk ke halaman "Transaksi", daftarnya tidak terpotong. Langkah 4: Publikasikan (Publish) Klik tombol biru Publikasikan (Publish) di pojok kanan atas. Masukkan alamat web yang Anda inginkan (misalnya: rafael-finance-app). Atur siapa yang bisa melihat situs ini. Jika ini untuk pribadi/keluarga, Anda bisa mengaturnya agar hanya orang tertentu yang bisa melihatnya, atau biarkan publik jika tidak masalah. Klik Publikasikan. Klik ikon panah ke bawah di sebelah tombol Publikasikan, lalu pilih Lihat situs yang dipublikasikan (View published site) untuk mencoba aplikasi Anda! Catatan Penting untuk Google Sites: Fitur "Simpan Transaksi" (menggunakan Local Storage) akan tetap berfungsi normal. Namun, karena Google Sites menggunakan sistem iframe (bingkai di dalam web), data transaksi Anda hanya akan tersimpan dan terlihat di browser dan perangkat yang sama saat Anda mengisinya. Seringkali fitur embed (sematkan) di Google Sites memang membatasi fungsi Local Storage demi keamanan, sehingga aplikasinya gagal berjalan normal. Mengingat aplikasi Anda saat ini menyimpan data menggunakan Local Storage (memori browser), ada satu fakta penting: data Anda hanya akan tersimpan di perangkat tempat Anda membukanya. Jadi, jika Anda membukanya di laptop, datanya tidak akan otomatis muncul di HP Anda, karena kita belum menggunakan database sungguhan. Karena alasan itu, sebenarnya ada cara yang 100% gratis, paling aman, dan sangat mudah tanpa perlu repot membuat website atau melakukan hosting sama sekali: Solusi 1: Jalankan Secara Offline (Lokal) Ini adalah cara terbaik untuk aplikasi keuangan pribadi, karena data Anda sama sekali tidak akan pernah dikirim ke internet. Sepenuhnya rahasia di komputer Anda. Pastikan kode terakhir yang saya berikan sudah Anda simpan dalam sebuah file bernama index.html. Klik kanan pada file index.html tersebut. Pilih Open With (Buka dengan), lalu pilih browser favorit Anda (Google Chrome, Firefox, Safari, atau Edge). Aplikasi akan langsung terbuka di browser Anda dan berfungsi 100% normal. Agar mudah diakses setiap hari, tekan tombol bintang (Bookmark) di browser Anda, dan letakkan di Bookmark Bar. Kapan pun Anda ingin mencatat keuangan, tinggal klik bookmark tersebut, bahkan saat komputer Anda tidak tersambung ke internet! Solusi 2: Gunakan Vercel (Jika Tetap Ingin Online) Jika Anda tetap ingin aplikasinya memiliki alamat link (URL) sendiri (misal agar bisa dibuka di perangkat lain meskipun datanya terpisah), alternatif terbaik selain Netlify adalah Vercel. Keunggulan: Sangat aman, digunakan oleh banyak developer profesional, gratis, dan tidak ada iklan. Cara pakai: Buat akun di vercel.com, buat proyek baru, lalu Anda tinggal men-drag-and-drop folder yang berisi file index.html Anda, persis seperti Netlify. Solusi 3: Gunakan GitHub Pages Ini adalah platform hosting statis gratis dari Microsoft (GitHub) yang sangat aman untuk jangka panjang. Keunggulan: Tidak akan pernah dihapus atau kadaluarsa. Kekurangan: Proses awalnya sedikit lebih teknis karena Anda harus membuat akun GitHub, membuat "Repository", dan mengunggah file HTML Anda ke sana sebelum halamannya bisa tayang. Apakah Anda ingin mencoba cara offline yang tidak memerlukan internet sama sekali ini, atau Anda tetap ingin panduan langkah demi langkah untuk mengunggahnya ke Vercel/GitHub? coba pakai github Pilihan yang sangat tepat! GitHub Pages adalah salah satu cara paling aman, 100% gratis selamanya, dan sangat diandalkan oleh para pembuat website di seluruh dunia. Karena Anda sudah memiliki file kodenya, kita tidak perlu menggunakan kode perintah (coding/terminal) yang rumit. Kita akan menggunakan cara unggah langsung dari browser (drag-and-drop). Berikut adalah langkah-langkah mudah mempublikasikan aplikasi Anda ke GitHub Pages: Langkah 1: Buat Akun GitHub Buka github.com dan buat akun gratis jika Anda belum memilikinya (klik Sign up). Ikuti proses pendaftarannya hingga selesai dan Anda masuk ke halaman utama (dashboard) GitHub. Langkah 2: Buat "Wadah" (Repository) Baru Di halaman utama GitHub, cari tombol hijau bertuliskan New atau Create repository (biasanya ada di sebelah kiri atas). Di kolom Repository name, ketik nama untuk aplikasi Anda, misalnya: rafael-finance. Pastikan Anda memilih opsi Public (ini wajib agar website Anda bisa diakses secara gratis). Gulir ke bawah dan klik tombol hijau Create repository. Langkah 3: Unggah File Anda Setelah repository dibuat, Anda akan melihat halaman dengan banyak teks kode. Jangan bingung, cukup cari dan klik tulisan biru berbunyi: "uploading an existing file" (tepat di bawah tulisan Quick setup). Anda akan dibawa ke halaman unggah. Sekarang, ambil file index.html yang berisi kode terakhir Anda di komputer, lalu tarik dan lepas (drag-and-drop) ke dalam kotak yang bertuliskan "Drag files here to add them to your repository". Tunggu hingga file selesai diunggah. Gulir ke bawah ke kotak Commit changes, lalu klik tombol hijau Commit changes. Langkah 4: Aktifkan Website (GitHub Pages) Sekarang file Anda sudah ada di dalam GitHub. Perhatikan deretan menu di bagian atas, klik menu Settings (ikon roda gigi). Di menu sebelah kiri, gulir ke bawah dan cari menu Pages. Klik menu tersebut. Di bagian Build and deployment, cari tulisan Branch. Ubah tombol dropdown yang bertuliskan "None" menjadi main (atau master). Klik tombol Save. Langkah 5: Selesai! Buka Website Anda Tunggu sekitar 1 hingga 3 menit (sistem GitHub sedang menyiapkan website Anda). Masih di halaman Pages tersebut, refresh (muat ulang) halaman browser Anda. Nanti akan muncul kotak di bagian atas dengan tulisan: "Your site is live at https://[username-anda].github.io/rafael-finance/". Klik link tersebut, dan aplikasi keuangan Anda sudah online secara aman! Sekarang Anda bisa membagikan link tersebut. Ingat, karena aplikasinya masih murni dan aman menggunakan Local Storage, data yang Anda catat di HP tidak akan muncul di Laptop, dan sebaliknya (tersimpan aman di masing-masing perangkat).
